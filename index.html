How to use

Arduino + RC522 reader (recommended on PC)

  1. Wire the MFRC522 module to your Arduino (SS → pin 10, RST → pin 9, 3.3V power).
  2. Install the MFRC522 library in Arduino IDE, then upload arduino/intern_rfid_reader/intern_rfid_reader.ino.
  3. Open this page in Chrome or Edge at localhost.
  4. Click Connect Arduino, pick the serial port, then use Read / Write with a card on the reader.

Phone NFC (Android)

  1. Switch to the Phone NFC tab.
  2. Open this page in Chrome on Android over HTTPS or localhost.
  3. Tap Read or Write, then hold the card near the phone.
